## 📝 Descrição do Projeto

Este projeto é uma **melhoria funcional e visual** baseada no site [ExifInfo.org](https://exifinfo.org/), uma ferramenta online que exibe os **metadados ocultos** embutidos em arquivos de imagem e mídia — como data de captura, localização GPS, modelo do dispositivo, configurações de câmera e muito mais.

A versão aprimorada, chamada **FindMe.AI**, mantém toda a base de acessibilidade do site original, adicionando novas funcionalidades voltadas para **usabilidade**, **personalização visual** e **versatilidade de formatos**.

---

## ✨ Melhorias Implementadas

### 🌗 Modo Escuro e Claro
- Alternância entre tema claro e escuro com um botão toggle
- Preferência salva para melhor experiência do usuário
- Contraste adequado em ambos os modos para garantir acessibilidade visual

### 🖼️ Upload de Imagens
- Interface para adicionar imagens diretamente pelo app
- Exibição dos metadados Exif extraídos da imagem (data, GPS, dispositivo, resolução, etc.)
- Suporte a formatos `.jpeg`, `.jpg`, `.png`, `.webp` e outros

### 📄 Conversor de PDF
- Funcionalidade de conversão integrada ao fluxo de análise
- Permite trabalhar com arquivos PDF além dos formatos de imagem tradicionais
- Extração de metadados de documentos PDF (autor, data de criação, ferramenta utilizada, etc.)

### ♿ Acessibilidade Mantida
- Estrutura semântica preservada do site original
- Navegação por teclado funcional
- Compatível com leitores de tela
- Alto contraste nos dois temas (claro e escuro)

## 🔎 O que são Metadados Exif?

**Exif (Exchangeable Image File Format)** é um padrão que define os formatos de tags de metadados em imagens, áudios e outros arquivos capturados por câmeras, celulares e dispositivos digitais.

Exemplos de metadados que podem ser extraídos:

- 📅 Data e hora de captura
- 📍 Coordenadas GPS (latitude e longitude)
- 📱 Fabricante e modelo do dispositivo
- 📷 Configurações da câmera (abertura, velocidade do obturador, ISO, flash)
- 👤 Nome do autor e informações de copyright
- 🗂️ Nome original do arquivo
